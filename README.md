# Glossário Interativo de Tags HTML

Este projeto é um guia prático e visual desenvolvido para a disciplina de Desenvolvimento de Sistemas. O objetivo é catalogar e demonstrar o funcionamento de diversas tags HTML, divididas por categorias funcionais, utilizando uma interface moderna com CSS Grid e Flexbox.

## Categorias Documentadas

O projeto organiza as tags nos seguintes grupos:

1. Marcadores: Tags de formatação de texto como mark, small, del, ins, sup e sub.
2. Semânticas: Tags que trazem significado ao conteúdo como abbr, cite, blockquote, time, address e q.
3. Interativas: Elementos de interação com o usuário como details, summary, progress, meter, textarea e button.
4. Estruturais: Organização de layout com header, main, footer, nav, figure e figcaption.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do conteúdo.
* CSS3: Estilização com foco em responsividade, posicionamento fixo (sidebar) e efeitos de transição (hover nos cards).

## Funcionalidades

* Menu Lateral Fixo: Navegação rápida entre as seções da página.
* Cards Informativos: Cada tag possui sua descrição e um exemplo prático de renderização.
* Design Responsivo: Layout em grade que se adapta ao conteúdo.

## Como Visualizar

1. Faça o download dos arquivos index.html e style.css.
2. Certifique-se de que ambos estão na mesma pasta.
3. Abra o arquivo index.html em qualquer navegador moderno.
